# 🚑 RapidAid
### Emergency Response and Resource Allocation System

---

## 📌 Overview
RapidAid is a simulation-based emergency response system designed to improve the allocation of critical healthcare resources such as ambulances and hospital beds.  
The system processes patient requests, prioritizes them based on urgency, and allocates the nearest available ambulance and hospital to minimize response time.

This project demonstrates the practical application of **Design and Analysis of Algorithms (DAA)** concepts to solve real-world emergency management problems.

---

## ⚙️ Key Features
- Patient request management  
- Priority-based emergency handling  
- Ambulance dispatch system  
- Hospital bed allocation  
- Shortest path calculation for ambulance routing  
- Waiting queue when resources are unavailable  
- Dynamic priority adjustment using Python  

---

## 🧠 Algorithms & Data Structures Used

| Concept | Application |
|------|------|
| Priority Queue | Processing patients based on priority |
| Dijkstra Algorithm | Finding shortest route between locations |
| Greedy Approach | Selecting nearest ambulance and hospital |
| Graph (Adjacency List) | Representing road network |
| Queue | Managing waiting patients |
