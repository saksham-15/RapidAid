Emergency Response and Resource Allocation System
#Overview
RapidAid is a simulation-based emergency response system designed to improve the allocation of critical healthcare resources such as ambulances and hospital beds. The system processes patient requests, prioritizes them based on urgency, and allocates the nearest available ambulance and hospital to minimize response time.
The project demonstrates the practical application of concepts from Design and Analysis of Algorithms (DAA) for solving real-world emergency management problems.
#Key Features
1.Patient request management
2.Priority-based emergency handling
3.Ambulance dispatch system
4.Hospital bed allocation
5.Shortest path calculation for ambulance routing
6.Waiting queue for patients when resources are unavailable
7.Dynamic priority adjustment using Python
